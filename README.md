# Customer Management System Using Hibernate

## Description

This project is a console-based Customer Management System developed in Java using Hibernate ORM for database interaction. It provides functionalities to register new customers, update customer details, delete customers, and query customer information using various attributes such as customer ID, mobile number, name, email, gender, or address.

The system leverages Hibernate to simplify database operations and improve maintainability through object-relational mapping. It’s designed for learning and demonstration purposes related to Java, Hibernate, and database CRUD operations.

---

## Features

- Customer registration (Create)
- Update customer email by customer ID (Update)
- Delete customer details by customer ID (Delete)
- Select customer details by customer ID (Read)
- Search customer details by mobile number
- Search customer details by name, email, gender, or address

---

## Technologies Used

- Java (JDK 8 or above)
- Hibernate ORM
- MySQL / any relational database (configure in `hibernate.cfg.xml`)
- Maven / Gradle (optional, based on your build tool)
- Console-based user interface
